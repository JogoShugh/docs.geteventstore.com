Class ConnectionSettings

A ConnectionSettings object is an immutable representation of the settings for an IEventStoreConnection. You can build a ConnectionSettings object using a ConnectionSettingsBuilder, either via the Create() method, or via the constructor of ConnectionSettingsBuilder.
